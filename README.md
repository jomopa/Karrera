# Karrera

Simulador arcade de carrera política española (2008–2030).

De panfletos a la Moncloa… o a tertuliano. Partidas de 5–8 minutos. Decisiones rápidas, eventos históricos, traiciones, memes y legado emergente.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
